Sheep Jump (WASM Edition) 🐑
This is a WebAssembly (WASM) demo of my upcoming game engine/library, Pygame++.

🚀 About this Project

This game serves as a technical showcase for Pygame++, a C++ library I am currently developing to bring Pygame-style simplicity to the C++ world. The full library is slated for release later this year.

This specific version was:

Written in C++ using the Pygame++ API.

Compiled into WebAssembly using Emscripten.

Optimized for browser-based play.

🎮 How to Play

Play the demo here: https://xlewis1.github.io/sheepjumppygame-wasm/

Objective: Jump over obstacles and keep the sheep moving!

Jump Left: Left Arrow Key

Jump Right: Right Arrow Key

Reset Game: Spacebar

Best Experience: Played on Google Chrome.

🛠️ Technical Details

The source code (main.cpp) and the core library headers (PygamePP.h) are currently kept private as the library is still in active development. This repository contains the build artifacts necessary to host the demo on the web.
